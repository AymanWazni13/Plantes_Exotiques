# Plantes_Exotiques
Plant Collection

Objectif du projet

Une site pour afficher une collection de plantes avec des informations détaillées provenant de l'API Trefle.io.

Prérequis

Node.js (version >= 18)

Installation et lancement

Cloner le dépôt :

   git clone 
   cd plant-collection

Installer les dépendances :

   npm install

Lancer l'application :

   npm start
